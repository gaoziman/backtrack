# 更新日志

本项目所有版本的更新内容记录于此。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## v0.8.0 - 2026-06-23

把"找回对话"升级为"主动收藏"：常用会话一键收藏并归类，统计面板焕然一新。

### 新增

- **会话收藏 + 自定义分类** — 在会话上点击星标即可收藏，或右键「收藏到分类」归入一个或多个自建分类。新增全屏收藏视图：左侧分类列表支持拖拽排序，右侧会话流支持子串搜索（CJK 友好）。会话行显示收藏星与分类色点，一眼辨认。收藏与分类独立持久化于本机，**绝不修改原始 jsonl**；删除分类时收藏降级为「未分类」而非丢失。

### 优化

- **使用统计仪表盘重设计** — 总会话升级为英雄卡并展示近 30 天趋势；工具占比新增环形图，月趋势补充数据点；活跃热力图改为固定紧凑小格并新增星期 / 月份坐标轴，告别拉伸过大的色块；最活跃目录由长进度条改为列对齐的排名榜单（序号 + 目录名 + 工具色点 + 占比）。卡片圆角统一为 8px。
- **全屏视图交互统一** — 收藏与统计两个全屏视图打开时自动隐藏侧边栏并互斥切换，选中会话即回到阅读视图。

## v0.7.0 - 2026-06-19

把"找回对话"升级为"沉淀知识"：会话摘要让你不必通读也能秒懂这次聊了什么。

### 新增

- **AI 会话摘要（可选，默认关闭）** — 阅读会话时，头部下方新增可折叠的「AI 摘要」块，调用你配置的 LLM 把整段对话概括为三段式：**一句话总结 / 关键结论 / 涉及代码**。手动触发、永久缓存、失败静默降级可重试；选中会话时自动回显已缓存的摘要（不触网）。复用 v0.6 的 AI 配置（地址 / 密钥 / 模型），与 AI 标题同一套设置。**默认关闭、密钥仅存本机、仅上传对话摘录（排除工具调用与输出）、绝不上传完整 jsonl。**

### 优化

- **README 隐私表述校正** — 明确「核心功能纯本地、AI 标题 / 摘要为可选且默认关闭」的边界，修正旧版"不调用任何 AI 模型"的绝对表述（功能自 v0.6 起即为可选触网，文档此前未同步）。

## v0.6.0 - 2026-06-18

会话辨认大升级：让标题说人话，让分支看得见。

### 新增

- **Codex 分支谱系可视化** — 阅读由 fork 衍生出的 Codex 会话时，头部出现「分支谱系」入口，弹出面板以缩进树展示整条 fork 谱系（祖先 → 当前 → 各分支），高亮当前会话，点击任意节点即可跳转阅读。仅在会话存在 fork 关系时显示。
- **智能标题提炼** — 升级标题派生逻辑：跳过 `$analysis` 等命令调用句，剥离 Markdown 链接 / 行内代码 / 路径等噪声，过滤 `<skill>`、`Base directory` 等系统注入文本；首句无意义时回退首条 AI 回复。纯本地启发式、不联网，实测唯一标题数提升约 5.5 倍，几乎消除「（无标题会话）」与大面积雷同标题。
- **AI 标题概括（可选，默认关闭）** — 为首句完全相同、启发式无法区分的会话，提供基于 LLM 的标题概括。需在「AI 标题设置」中自行填写 API 地址、密钥与模型并主动开启；按需触发、永久缓存、失败静默降级。标题优先级为：用户自定义 > AI 概括 > 启发式派生。**默认关闭、纯本地；密钥仅存本机、不上传、不入代码库。**

### 优化

- **标题逻辑版本号** — 标题提炼逻辑升级后，首次启动自动重建一次索引，旧会话标题随之更新；用户手动重命名的标题始终保留。

## v0.5.0 - 2026-06-17

补上「找回对话 → 带走」的闭环出口，并打磨搜索体验。

### 新增

- **单会话导出 Markdown / HTML** — 右键会话 →「导出…」或在阅读器头部点「导出」，把整段对话保存为 `.md`（适合笔记/文档）或自包含 `.html`（双击即看、离线可用）。可选是否包含工具调用与输出；通过系统「另存为」对话框自选保存位置；导出后可一键在 Finder 中定位。**纯导出、HTML 内容已转义、绝不修改原始 jsonl。**
- **搜索过滤器补全** — 新增「按目录」过滤（搜索区目录下拉，仅在指定目录内检索）；「按工具」过滤下沉到后端，结果更准、不再受 200 条上限挤占。配合已有的时间/角色过滤，四维度任意组合。

### 优化

- **搜索结果列表重构** — 命中条目改为块条目样式：独立圆角块、hover 描边、命中关键词由实心色块收敛为变色加下划线，工具标识弱化为小方点，整体更清爽易扫读。

### 修复

- **resume 命令转义加固** — 一键终端恢复时，会话 id 经 shell 单引号安全转义，含特殊字符时不再有命令注入隐患；正常会话行为不变。

## v0.4.0 - 2026-06-17

会话整理增强：标题不再只能将就系统派生值。

### 新增

- **会话标题重命名** — 右键会话 →「重命名标题」，给会话起个好记的名字。自定义标题**独立持久化、不会被增量重索引覆盖、绝不修改原始 jsonl**；弹窗内可一键「恢复默认标题」。

## v0.3.0 - 2026-06-17

实时化：从「手动刷新」进化为「文件变更自动跟进」。

### 新增

- **文件变更监听，自动刷新** — 常驻监听 `~/.claude/projects` 与 `~/.codex/sessions`，新会话产生 / 文件变更 / 删除后，自动增量索引并**静默刷新**左栏列表，无需手动点扫描。防抖合并写入风暴，全程不打断当前阅读（不弹提示、不改选中会话与展开态）。

## v0.2.1 - 2026-06-17

性能优化：启动索引改为增量，告别每次启动的长时间全量重扫。

### 优化

- **启动增量索引** — 仅重新解析有改动的会话文件，未变化的跳过、已删除的清理；首次升级会重建一次，**之后每次启动近乎瞬时**（实测复扫约 28 秒 → 0.4 秒）。

## v0.2.0 - 2026-06-17

搜索体验大幅升级：从"能搜到会话"进化为"看得见命中、跳得过去"。

### 新增

- **命中片段预览** — 搜索结果在标题下直接展示包含关键词的正文片段并高亮，不点开即可判断。
- **会话内查找（⌘F）** — 打开会话后按 ⌘F 在对话内逐个查找跳转，命中高亮、计数、上一个/下一个，长消息自动展开。
- **搜索过滤器** — 按时间范围（近 7 天 / 近 30 天）与角色（我 / AI）缩小搜索结果。
- **FTS5 全文检索引擎** — 引入 trigram 分词，支持相关度排序；中文短词（1-2 字）自动走子串兜底，CJK 检索不退化。
- **搜索态目录折叠** — 搜索结果的目录分组支持点击折叠 / 展开。

### 优化

- **索引体积治理** — 全文索引仅覆盖对话正文、排除工具调用输出并收紧单条上限，索引体积大幅下降，启动更轻。

## v0.1.0 - 2026-06-16

首个公开版本。Backtrack 是一个 macOS 桌面应用，把本地 **Claude Code** 与 **Codex** 的历史会话按目录归类，支持全文搜索、完整对话阅读与一键终端 `resume`。纯本地、只读、不联网、不调用任何 AI 模型。

### 新增

- **双工具扫描** — 自动索引 `~/.claude/projects` 与 `~/.codex/sessions` 下的全部会话。
- **目录为主轴** — 左栏按工作目录（cwd）聚合，Claude / Codex 混排并以彩色标签区分。
- **完整对话阅读器** — user/assistant 气泡、工具调用可折叠、Markdown 与代码高亮。
- **全文搜索** — 跨所有会话的子串搜索，CJK 友好，命中即点即达。
- **一键恢复** — 复制 `claude --resume <id>` / `codex resume <id>`，或唤起 iTerm / Terminal / Warp 自动 `cd` 并执行。
- **整理而不破坏** — 隐藏、关注置顶；删除走废纸篓（可恢复），绝不修改原始 jsonl。
