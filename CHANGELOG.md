# 更新日志

本项目所有版本的更新内容记录于此。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## v0.5.0 - 2026-06-17

补上「找回对话 → 带走」的闭环出口。

### 新增

- **单会话导出 Markdown / HTML** — 右键会话 →「导出…」或在阅读器头部点「导出」，把整段对话保存为 `.md`（适合笔记/文档）或自包含 `.html`（双击即看、离线可用）。可选是否包含工具调用与输出；通过系统「另存为」对话框自选保存位置；导出后可一键在 Finder 中定位。**纯导出、HTML 内容已转义、绝不修改原始 jsonl。**

## v0.4.0 - 2026-06-17

会话整理增强：标题不再只能将就系统派生值。

### 新增

- **会话标题重命名** — 右键会话 →「重命名标题」，给会话起个好记的名字。自定义标题**独立持久化、不会被增量重索引覆盖、绝不修改原始 jsonl**；弹窗内可一键「恢复默认标题」。

## v0.3.0 - 2026-06-17

实时化：从「手动刷新」进化为「文件变更自动跟进」。

### 新增

- **文件变更监听，自动刷新** — 常驻监听 `~/.claude/projects` 与 `~/.codex/sessions`，新会话产生 / 文件变更 / 删除后，自动增量索引并**静默刷新**左栏列表，无需手动点扫描。防抖合并写入风暴，全程不打断当前阅读（不弹提示、不改选中会话与展开态）。

## v0.2.1 - 2026-06-17

性能优化：启动索引改为增量，告别每次启动的长时间全量重扫。

### 优化

- **启动增量索引** — 仅重新解析有改动的会话文件，未变化的跳过、已删除的清理；首次升级会重建一次，**之后每次启动近乎瞬时**（实测复扫约 28 秒 → 0.4 秒）。

## v0.2.0 - 2026-06-17

搜索体验大幅升级：从"能搜到会话"进化为"看得见命中、跳得过去"。

### 新增

- **命中片段预览** — 搜索结果在标题下直接展示包含关键词的正文片段并高亮，不点开即可判断。
- **会话内查找（⌘F）** — 打开会话后按 ⌘F 在对话内逐个查找跳转，命中高亮、计数、上一个/下一个，长消息自动展开。
- **搜索过滤器** — 按时间范围（近 7 天 / 近 30 天）与角色（我 / AI）缩小搜索结果。
- **FTS5 全文检索引擎** — 引入 trigram 分词，支持相关度排序；中文短词（1-2 字）自动走子串兜底，CJK 检索不退化。
- **搜索态目录折叠** — 搜索结果的目录分组支持点击折叠 / 展开。

### 优化

- **索引体积治理** — 全文索引仅覆盖对话正文、排除工具调用输出并收紧单条上限，索引体积大幅下降，启动更轻。

## v0.1.0 - 2026-06-16

首个公开版本。Backtrack 是一个 macOS 桌面应用，把本地 **Claude Code** 与 **Codex** 的历史会话按目录归类，支持全文搜索、完整对话阅读与一键终端 `resume`。纯本地、只读、不联网、不调用任何 AI 模型。

### 新增

- **双工具扫描** — 自动索引 `~/.claude/projects` 与 `~/.codex/sessions` 下的全部会话。
- **目录为主轴** — 左栏按工作目录（cwd）聚合，Claude / Codex 混排并以彩色标签区分。
- **完整对话阅读器** — user/assistant 气泡、工具调用可折叠、Markdown 与代码高亮。
- **全文搜索** — 跨所有会话的子串搜索，CJK 友好，命中即点即达。
- **一键恢复** — 复制 `claude --resume <id>` / `codex resume <id>`，或唤起 iTerm / Terminal / Warp 自动 `cd` 并执行。
- **整理而不破坏** — 隐藏、关注置顶；删除走废纸篓（可恢复），绝不修改原始 jsonl。
